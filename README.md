# 15 Pencils (Tkinter)

A small two-player graphical game built with Python's built-in **Tkinter** library.

## Rules

- The game starts with **15 pencils** on screen.
- Players alternate turns.
- On each turn, a player can remove **1, 2, or 4** pencils.
- The player who takes the **last** pencil **loses**.

## Run

```bash
python3 hello.py
```
